# Hacking_Toolbox

A personal collection of offensive security tools, payloads, and scripts built up over years of pentesting and bug bounty.

The underlying goal is always the same: have the right payload or script ready when needed, without digging through old notes or rewriting from scratch.

## Legal

All tools and payloads in this repository are intended for use on systems you own or have explicit written permission to test. Unauthorized use is illegal.
